Cats & Dungeons

Cats & Dungeons é um jogo roguelike desenvolvido em Java, com exploração procedural, combate estratégico e progressão baseada em ondas. Utiliza pathfinding A*, colisão AABB, física vetorial e inventário estruturado em arrays. Estruturas especiais, puzzles e chefes surgem em fases avançadas, trazendo variedade e desafios crescentes conforme o jogador evolui.

JAVA TinySound JSON A* Pathfinding Física 2D AABB Collision Arrays Game Loop Roguelike

Reconhecimento de Gestos • Teachable Machine

Projeto que integra um modelo do Teachable Machine com uma aplicação web, capturando gestos via webcam e exibindo previsões em tempo real. Inclui uma área de testes para avaliar estabilidade das predições e um mini-jogo educativo onde o usuário imita animais; imagens são carregadas dinamicamente via API protegida.

Guardian of Wynn • Discord Bot

Bot avançado para Discord, focado em moderação, automação e monitoramento da guilda do Wynncraft. Possui backend robusto com requisições constantes à API para acompanhar atividades e aplicar sistema de pontos. Inclui logs avançados de mensagens apagadas/editadas, eventos e sorteios. Executa rotinas agendadas para tarefas automáticas. Utiliza slash commands organizados em módulos e gerenciamento de embeds complexos.

Node.js JavaScript Discord.js Slash Commands JSON Event System API Integration Logs & Transcripts

Classificação de Câncer de Mama

Implementação completa de uma pipeline de Machine Learning para prever tumores benignos (0) e malignos (1) no dataset Breast Cancer Wisconsin. O projeto foca principalmente em modularidade, reprodutibilidade e análise aprofundada de modelos clássicos, comparando seu desempenho através de métricas robustas como F1-Score e ROC-AUC. Inclui análise detalhada de erros (FP vs FN) e importância de features.

Python Scikit-learn Pandas Seaborn Classificação Binária Joblib Análise de Erros
