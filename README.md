## Visão Geral

Este projeto consiste em um portfólio pessoal desenvolvido com tecnologias web fundamentais (Vanilla Stack). A plataforma foi projetada para atuar como uma vitrine digital, apresentando minha evolução no desenvolvimento de software, projetos acadêmicos e perfil pessoal.

O foco principal do projeto é a **performance e simplicidade**, utilizando uma arquitetura leve sem dependências de frameworks externos, resultando em um carregamento instantâneo e uma interface fluida.

## Arquitetura de Conteúdo

O portfólio está organizado em três pilares fundamentais:

* **Início:** Uma introdução sobre quem sou, minhas preferências na área tecnológica e uma visão geral das minhas contribuições no ecossistema GitHub.
* **Projetos:** Exposição de projetos pessoais com detalhamento das tecnologias aplicadas, documentação e links para acesso direto.
* **Sobre Mim:** 
    * **Trajetória:** Cronologia desde o início dos estudos até a formação acadêmica atual e cursos complementares.
    * **Habilidades Técnicas:** Stack tecnológica atual e ferramentas em processo de aprendizado.
    * **Interesses & Hobbies:** Uma visão humanizada sobre atividades extracurriculares, literatura, cinema e séries.

## Diferenciais Técnicos

* **Vanilla Stack:** Desenvolvimento puramente em HTML5 semântico, CSS3 e JavaScript (ES6+), demonstrando domínio nos fundamentos da web.
* **Interface Clean:** Design focado na experiência do usuário e legibilidade.
* **Performance Otimizada:** Baixo consumo de recursos e alta pontuação em métricas de Core Web Vitals devido à ausência de bibliotecas pesadas.
* **Deploy Contínuo:** Hospedagem e automação de entrega via Vercel.

## Especificações

* **Desenvolvedor:** Francisco Neto
* **Tecnologias:** HTML5, CSS3, JavaScript.
* **Hospedagem:** Vercel (Edge Network).
